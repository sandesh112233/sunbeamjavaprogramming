Introduction to web development

Web development is the process of creating websites and web applications that run on the internet.It involves designing,coding, and maintaining web pages using various technologies.

Objectives of web Development

  • Understand the basic structure of HTML document
  • Learning formatting and text aligning
  • Use lists images and hyperlinks

Core concepts of a Web page

  1. HTML
    • Defines structure of the pages
    • Uses tags like < html >,< head >,< body >
  2. CSS
    • Used for styling the web pages
    • Controls layout and colors
  3. JavaScript
    • Adds interactivity
    • Handles events and validations

Text formatting Options

HTML allows text to be bold, italic ,underlined ,and even colored to make the content attractive.

Special Characters

Some special symbols used in HTML are:

  • Copyright symbol: ©
  • Registered trademark: ®
  • Trademark: &trade
  • Less than sign: <
  • Greater than sign: >
  • Ampersand: &

Hyperlinks in HTML

Web pages can be linked using anchor tags.
